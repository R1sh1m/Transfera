# Security Policy

## Supported Versions

Only the latest release (v2.4.0 and above) receives security fixes. Older versions should be updated as soon as possible.

## Reporting a Vulnerability

Please report vulnerabilities privately via the **Report a vulnerability** button on the [Security tab](https://github.com/R1sh1m/Transfera/security) — do not open a public issue.

## Code Signing

Windows release binaries are code-signed via [SignPath Foundation](https://about.signpath.io/) (pending approval). The certificate is issued to **SignPath Foundation**. To verify, right-click the installer → Properties → Digital Signatures.

## Response Timeline

This project is maintained by a single developer on a best-effort basis. I aim to acknowledge reports within 7 days. Fix timelines depend on severity and availability.
